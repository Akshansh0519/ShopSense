System Design

How It Works

A decoupled MLOps pipeline that trains offline and serves in real-time through a secured FastAPI microservice.

01
⬡

Data Layer

H&M transaction logs are loaded via Pandas, encoded into sparse CSR interaction matrices, and split temporally to prevent data leakage.

PandasSciPy CSRTemporal Split
—
02
◈

Model Training

Three parallel learners — ALS (matrix factorization), BPR (pairwise ranking), and Sentence-Transformer NLP embeddings — are trained and checkpointed via MLflow.

ALSBPRNLPMLflow
—
03
◇

Hybrid Fusion

Scores from all models are normalized and fused with configurable weights. MMR reranking then diversifies the final list to reduce visual redundancy.

Score FusionMMRDiversity
—
04
△

API Serving

FastAPI with API key auth, rate limiting, CORS protection, and Redis cache serves ranked recommendations at sub-50ms with cold-start fallback.
